== Description ==

Send notification to admin when someone tracking website. When there are new articles will be sent an email to the email signed in tracking page.

Ask for the latest version of wp (minimum 4.3 ).

Requires PHP smallest is 5.2

You need to install SMTP for email admin you used to send mail

== Installation ==

1. Upload the entire `subscriber-website` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Show in Appearance -> Subscriber Website.
